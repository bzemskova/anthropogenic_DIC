# anthropogenic_DIC
Data and code to accompany manuscript titled "Changes in the Oceanic Carbon Storage due to Anthropogenic Carbon Input over the Past Three Decades" by F. Hong and V.E. Zemskova

#### "raw_Cstar_data" folder
HDF5 files of $C^*$ at each grid point for $1^\circ$ horizontal resolution for each year over the 1992-2022 analysis period

#### "processed_data" folder
HDF5 files of computed $C^*$ trends and other helpful variables (see folder README)

#### "analysis" folder
Jupyter notebooks with code to post-process and plot data presented in the manuscript

#### "figures" folder
Figures corresponding to figures in the manuscript made using Jupyter notebooks in the "analysis" folder
